# CareerGraph - Plataforma de Orientação de Carreira para Redução de Desiguadades
## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de contribuir para o Objetivo de Desenvolvimento Sustentável 10 (ODS 10) da ONU - Redução das Desigualdades.

O Censo da Educação Superior, no ano de 2022, revelou que apenas 24,2% dos jovens de 18 a 24 acessam o ensino superior. Levando esse problema em consideração, a falta de acesso à informação sobre cursos, carreiras e oportunidades acadêmicas é um dos muitos fatores limitadores que contribuem para essa questão. Tendo isso em mente, o projeto CareerGraph vem à tona com o objetivo de transformar o acesso à esse tipo de informação em algo simples, eficiente e de fácil acesso a todo e qualquer tipo de pessoa.
